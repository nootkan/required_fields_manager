# Security Policy

## Supported Versions

Only the latest public release of this project is actively supported with security updates.

| Version | Supported |
| ------- | --------- |
| Latest release | Yes |
| Older releases | No |

## Reporting a Vulnerability

If you discover a security vulnerability, please do not open a public GitHub issue.

Instead, report it privately by emailing:

webmaster@vanislebc.com

Please include:

- A description of the vulnerability
- Steps to reproduce the issue
- The affected file, feature, or plugin version if known
- Any screenshots or proof-of-concept details that may help confirm the issue

I will review valid security reports as soon as possible and, if confirmed, work on a fix before publicly discussing the issue.
